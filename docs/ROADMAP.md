Roadmap
=======

M1: Auth + Skeleton (Done)
--------------------------
- Login/logout, session, JSON user storage
- Basic dashboard and health endpoint
- Systemd + Nginx deployment

M2: Core Analysis Foundations
----------------------------
- Add `app/core/seo_analyzer.py` interface and a simple page analyzer
- Results storage and listing
- Basic API for trigger + status

M3: Specialized Analyzers
-------------------------
- Intent, Schema planner, Keyword clustering (minimal viable)
- HTML/CSV/JSON report rendering

M4: Performance and UX
----------------------
- Threaded job runner with a simple queue
- Progress UI and better dashboard

M5: Security and Ops
--------------------
- CSRF, rate-limiting, password reset/change
- Better logs/metrics and error reporting
