ğŸ”§ Common Issues

"redirect_uri_mismatch" Error

Cause: The redirect URI in the request doesn't match what's registered in Google Cloud Console.

Solution:

  • Go to Google Cloud Console â†’ Credentials
  • Edit your OAuth 2.0 Client ID
  • Make sure this exact address is registered: http://seoanalyze.shahinvaseghi.ir/search-console/oauth2callback
  • No trailing slash!

"Access blocked" Error

Cause: OAuth consent screen is not properly configured.

Solution:

  • Go to OAuth consent screen
  • Add your email to Test users
  • Make sure Publishing status is "Testing"

"API has not been enabled" Error

Cause: Search Console API is not enabled.

Solution:

  • Go to APIs & Services â†’ Library
  • Find Search Console API
  • Click Enable
  • Wait a few minutes and try again
